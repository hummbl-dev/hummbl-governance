## Methodology

- Repo inventory: `gh repo list hummbl-dev --limit 200` via GitHub API
- Artifact check: `gh api repos/hummbl-dev/<repo>/contents/<artifact>` for each of 13 artifacts
- Scoring: 15-point scale (13 checked artifacts + HUMMBL_FORK.md + 1 reserved for scripts/validate.*)
- HUMMBL_FORK.md counted as "missing" for non-forks per 2026-06-22 convention (N/A but scored)
- All 104 repos appear exactly once in the audit
